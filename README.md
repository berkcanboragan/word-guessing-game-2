<h1>WORD GUESSING GAME 2</h1>

<p>This is another simple Wordle-like word guessing game built with vanilla JavaScript, HTML, and CSS.
The game randomly selects a Turkish word and challenges the player to guess it within 6 attempts.</p>

<h3>How It Works</h3>
<ul>
  <li>A random word is chosen from a predefined Turkish word list.</li>
  <li>The number of input boxes is generated dynamically based on the word length.</li>
  <li>The player types one letter per box.</li>
  <li>Press Enter to submit a guess.</li>
  <li>The game ends when the word is guessed correctly or the player reaches 6 unsuccessful attempts</li>
</ul>

<h3>Build With</h3>
<ul>
  <li>HTML</li>
  <li>SCSS</li>
  <li>JavaScript</li>
  <li>Bootstrap</li>
</ul>

<h3>Notes</h3>
<ul>
  <li>All logic is handled with pure JavaScript.</li>
  <li>Designed for learning and practicing DOM manipulation and game logic.</li>
</ul>
